# Repositorio de Base de Datos — MecanicWeb

Este repositorio contiene todo lo relacionado con la estructura y la lógica de la información de nuestro proyecto **"Mecanics Web"**.

## Versión actual
**v0.5 Estructura inicial** - Actualizado el 14 de noviembre de 2025
**Responsable de Actualización:** Jariff Ramiro Argaez Mendez

## Componentes
Para que sea fácil encontrar las cosas, organizamos el repositorio así:

**`/Modelado/`**: Aquí están los diagramas visuales (MER/DER) que muestran cómo se relaciona la información.

  `/Diagrama_logico/`: El diseño logico relacional de las tablas de datos con sus tipos de datos, restricciones, etc.
  
  `/Diagrama_mer/`: El diseño del diagrama de entidad-relacion.
  
**`/SQL/`**: Contiene los archivos de código (`.sql`) que se usan para crear la base de datos y para meter los datos iniciales.
